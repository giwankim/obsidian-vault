> Curated collection of graphics programming resources from the GPVM meetup, covering tutorials (Learn OpenGL, VkGuide, PBRT), math references, ray tracing papers, and tools across OpenGL, Vulkan, WebGPU, and software rendering.

## Resources

This page contains a collection of graphics programming and adjacent resources curated by our meetup attendees.

New to graphics programming and don't know where to start? Check our curated resources.
